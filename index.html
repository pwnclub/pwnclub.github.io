Just Type

  • Offline typing speed test application
  • Solo project taking some inspiration from 10FastFingers

  • Built with Tkinter using Python
  • https://github.com/pwnclub/just-type


Kinect Four

  • Leap Motion controlled Connect Four versus an AI
  • Worked on project with a team at Hack Western 4
  • Helped with backend (Python) and frontend (Javascript), as well as all art
